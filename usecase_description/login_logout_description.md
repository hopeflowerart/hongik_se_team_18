# 로그인

| Actor Action | System Response |
|---|---|
|  | 1. 로그인 화면을 보여준다. |
| 2. 사용자는 ID와 비밀번호를 입력한다. |  |
| 3. 사용자는 로그인 버튼을 클릭한다. |  |
|  | 4. 메인 화면을 보여준다. |
| 로그인 실패 시 |  |
|  | 4a. 오류 메시지를 보여준다. |

# 로그아웃

| Actor Action | System Response |
|---|---|
| 1. 사용자는 로그아웃 버튼을 클릭한다. |  |
|  | 2. 로그아웃 확인 팝업을 보여준다. |
| 3. 사용자는 확인 버튼을 클릭한다. |  |
|  | 4. 로그인 화면으로 이동한다. |
| 로그아웃 취소 시 |  |
| 3a. 사용자는 취소 버튼을 클릭한다. |  |
|  | 3b. 기존 화면을 유지한다. |
