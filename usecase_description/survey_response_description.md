설문 응답

|                         Actor Action                         |                 System Response                 |
|--------------------------------------------------------------|-------------------------------------------------|
| 1. 사용자가 설문 상세 정보 화면에서 응답하기 버튼을 클릭한다.              |                                                 |
|                                                              | 2. 설문 응답 화면으로 이동한다.                       |
| 3. 사용자가 설문 문항에 대한 응답을 입력한다.                          |                                                 |
| 4. 사용자가 응답 버튼을 클릭한다.                                   |                                                 |
