# 응답한 설문 조회

| Actor Action | System Response |
|---|---|
| 1. 사용자는 응답한 설문 조회 메뉴를 선택한다. |  |
|  | 2. 응답한 설문 리스트를 보여준다. |
|  | 3. 설문 제목, 응답 내용, 응답 일시를 보여준다. |
| 응답 기록이 없는 경우 |  |
|  | 3a. 응답 기록 없음 메시지를 보여준다. |

---

# 응답한 설문 수정

| Actor Action | System Response |
|---|---|
| 1. 사용자는 수정할 설문을 선택한다. |  |
|  | 2. 기존 응답 내용을 보여준다. |
| 3. 사용자는 응답 내용을 수정한다. |  |
| 4. 사용자는 수정 버튼을 클릭한다. |  |
|  | 5. 수정 완료 메시지를 보여준다. |
| 설문이 종료된 경우 |  |
|  | 5a. 수정 불가 메시지를 보여준다. |

---

# 응답한 설문 삭제

| Actor Action | System Response |
|---|---|
| 1. 사용자는 삭제할 설문을 선택한다. |  |
|  | 2. 삭제 확인 팝업을 보여준다. |
| 3. 사용자는 확인 버튼을 클릭한다. |  |
|  | 4. 삭제 완료 메시지를 보여준다. |
| 설문이 종료된 경우 |  |
|  | 4a. 삭제 불가 메시지를 보여준다. |
